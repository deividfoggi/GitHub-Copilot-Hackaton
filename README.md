# GitHub Copilot Hackathon

Este repositório hospeda o workshop prático de GitHub Copilot + Microsoft Foundry,
usando um sistema de Boletim de Ocorrência como base para evolução incremental.

## O desafio

O desafio do hackathon é transformar uma aplicação funcional, mas simples,
em uma solução mais inteligente e pronta para evolução contínua.

Ponto de partida:

- Um sistema de registro de boletins de ocorrência, com uma interface básica e regras fixas.
- O sistema já registra boletins e classifica ocorrências com regras fixas.

Meta do desafio:

- Evoluir o código com apoio do GitHub Copilot.
- Introduzir inteligência com um agente do Microsoft Foundry.
- Integrar essa inteligência ao fluxo real da aplicação.

Resultado esperado:

- Um fluxo completo em que o participante entende, modifica, integra e melhora
  o sistema, saindo com uma base técnica prática para projetos reais.

## Comece por aqui

1. Acesse o guia principal do workshop em [bo-app/workshop/README.md](bo-app/workshop/README.md).
2. Siga as etapas na ordem proposta (1 a 5).
3. Use os arquivos de apoio na pasta [bo-app/workshop](bo-app/workshop).

## O que você vai vivenciar

- Entender um sistema real com apoio do GitHub Copilot.
- Evoluir funcionalidades de ponta a ponta com segurança.
- Criar e integrar um agente do Microsoft Foundry ao fluxo da aplicação.
- Encerrar com otimização técnica (testes, qualidade e organização).
